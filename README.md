# Неоновый Кибер-Город ночью / Neon Cyber-City at Night

OpenGL проект, создающий атмосферный киберпанк город с неоновым освещением.

## Особенности

- **Высокие небоскрёбы** с процедурными окнами и неоновой подсветкой
- **Летающие машины** с фарами и задними огнями
- **Голограммы** на анимированных рекламных щитах
- **Мокрый асфальт** с отражениями (Fresnel эффект)
- **Много цветных неоновых источников света** (point + spot)
- **Анимированные рекламные щиты** с различными паттернами
- **Мигающие огни** на зданиях и дронах
- **Пар из канализации** (партикл-система)
- **Пролетающие дроны** с blinking LED
- **Сильный bloom и glow эффекты** (post-processing)
- **Камера** может летать между зданиями или ехать по улице
- **2D интерфейс** с футуристическим дизайном и сканлайнами

## Управление

| Клавиша | Действие |
|---------|----------|
| W/S | Движение вперёд/назад |
| A/D | Движение влево/вправо |
| Space/Ctrl | Вверх/вниз (в режиме полёта) |
| Мышь | Обзор камеры |
| C | Переключение режимов полёт/ходьба |
| B | Переключение bloom эффекта |
| U | Переключение UI |
| ESC | Выход |

## Требования

- Windows 10/11
- Visual Studio 2019 или новее
- Видеокарта с поддержкой OpenGL 3.3+
- **Никаких внешних библиотек не требуется** - используется только стандартный OpenGL из Windows SDK

## Установка

1. Откройте `creative_project.sln` в Visual Studio
2. Сборка проекта (Ctrl+Shift+B)
3. Запуск (F5)

## Технические детали

### Математика
- Все математические функции (vec3, mat4) реализованы вручную
- Замена библиотеки GLM на собственные структуры и функции

### Окно и ввод
- Создание окна через Win32 API (замена GLFW)
- Обработка клавиатуры и мыши через сообщения Windows

### OpenGL
- Загрузка расширений через wglGetProcAddress (замена GLEW)
- Создание контекста OpenGL 3.3 Core Profile через wglCreateContextAttribsARB

### Шейдеры
- Vertex/Fragment шейдеры для зданий, машин, дронов, билбордов
- Post-processing шейдеры для bloom эффекта (extract → blur horizontal → blur vertical → composite)
- UI шейдер для scanline эффекта

### Рендеринг
- Framebuffer Object (FBO) для post-processing эффектов
- Depth testing и blending для правильной видимости объектов

### Процедурная генерация
- Случайные позиции и размеры зданий
- Различные паттерны окон (grid, flickering, strips)
- Анимированные billboard паттерны (scrolling bars, pulsing circles, checkerboard, rainbow)
